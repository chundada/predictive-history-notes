## 第一部分：博弈论基础（约10张）

### Slide 2: 为什么学博弈论？
- 成为更好的人（思考力+道德+想象力）
- 理解世界（新闻不再是噪音）
- 预测未来（掌握自己命运）

### Slide 3: 五大传统理论 vs 博弈论
- 宗教（善恶）❌ · 生物学（性）❌ · 种族 ❌ · 经济（钱）❌ · 自由主义（进步）❌
- **博弈论：Players + Rules + Incentives → 预测** ✅

### Slide 4: 博弈三要素
```
Players（玩家）→ 谁在玩？
   +
Rules（规则）→ 边界条件是什么？
   +
Incentives（激励）→ 如何赢？
   ↓
Convergence Point = The Game
```

### Slide 5: 社会三指标
- **能量 Energy**：工作努力度、专注度
- **开放 Openness**：学习意愿、承认错误
- **凝聚力 Cohesion**：团队意识、为彼此牺牲
- 三者皆高=崛起 · 三者皆低=衰落

### Slide 6: 超级结构决定一切
```
人口 · 财富 · 技术 · 竞争 → Superstructure
        ↓
    决定博弈类型
        ↓
    决定玩家行为
```

### Slide 7: 核心洞察 #1-3
- **地位 > 金钱**（地位是零和博弈）
- **制定规则者永远赢**
- **被邀请玩别人的游戏 = 你会输**

## 第二部分：四大定律（约8张）

### Slide 8: 非对称性定律
- 强者优势随成功 → 劣势
- 三大优势 → 三大劣势：
  - 人口 → 不平等
  - 组织 → 寄生精英
  - 深度 → 自满
- **越强大越脆弱**

### Slide 9: 升级定律
- **控制 > 支配**
- 灵活性 > 力量
- 校园霸凌实验：新人靠不回应让霸凌者自我毁灭

### Slide 10: 末世论收敛定律
- **胜利 = 质量 × 能量 × 协调**
- 协调最重要 = 质量的4倍
- 七大宗教体系 → 末世叙事惊人趋同
- 六大收敛点 → 可预测的地缘政治

### Slide 11: 近因定律
- 你在同时玩多个游戏
- **最接近你的游戏影响最大**
- 国家外交 = 国内政治的延伸
- 内部冲突 → 决定外部行为

## 第三部分：文明生命周期（约6张）

### Slide 12: 八阶段循环
```
Startup → Hereditary → Elite Overproduction
→ Warring States (黄金时代) → Intermarriage
→ Secret Societies → Mercenaries → Takeover
→ 循环重来
```

### Slide 13: 历史铁律
- **最弱者总是征服最强者**
- 秦灭六国 · 马其顿灭希腊 · 罗马灭希腊化诸国
- 阿卡德灭苏美尔 · 西班牙500人灭阿兹特克
- Ibn Khaldun: Asabiyyah（凝聚力）

### Slide 14: 革命的三大触发
1. **债务**：利息累积 → 无力偿还
2. **奴隶化**：身体成抵押 → 世代为奴
3. **土地丧失**：失去生存基础

### Slide 15: 革命 = "have a lot vs have some"
- 中国革命 = 城市精英 vs 农村精英
- 商人/中产永远领导革命
- 动员公式："跟我来，免债+分地+废奴"

## 第四部分：全球博弈体系（约8张）

### Slide 16: 英国奠基（#6）
- 英格兰银行(1694)：**利润私有化，损失社会化**
- 三支柱：金融中心 + 软实力 + 皇家海军

### Slide 17: 美国扩展（#7）
- 解决英国三重局限：种族→移民、金→石油美元、地理→大陆
- 美国 = 国家即游戏：开放 + 公平 + 清晰
- 宪法 = 让游戏高于权力

### Slide 18: 美元体系
- 1944布雷顿森林 → 1971尼克松冲击
- 石油美元 + 中国 = 全球对美元上瘾
- 价格等级：资源→制造→知识→金融

### Slide 19: 游戏重置（Game Reset）
- 2008中国救市 → 中国要座位 → 美国说"不"
- 2016特朗普 → 贸易战
- 2022俄罗斯 → 说"去你的价格等级"
- **我们处于Game Reset时代**
